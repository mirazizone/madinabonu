# Инструкция по созданию симлинка для storage

Для того, чтобы изображения учителей и курсов отображались на сайте, нужно создать симлинк.

## Способ 1: Через командную строку (от имени администратора)

1. Откройте PowerShell или CMD от имени администратора
2. Перейдите в папку проекта:
   ```
   cd C:\OSPanel\home\Madinabonu
   ```
3. Выполните команду:
   ```
   php artisan storage:link
   ```

## Способ 2: Через PowerShell (от имени администратора)

1. Откройте PowerShell от имени администратора
2. Перейдите в папку проекта:
   ```
   cd C:\OSPanel\home\Madinabonu
   ```
3. Выполните команду:
   ```
   New-Item -ItemType SymbolicLink -Path "public\storage" -Target "storage\app\public" -Force
   ```

## Способ 3: Вручную (если есть доступ к OSPanel)

Если у вас есть доступ к PHP через OSPanel, выполните команду через веб-интерфейс или используйте полный путь к PHP:
```
C:\OSPanel\modules\php\PHP_8.3\php.exe artisan storage:link
```

После создания симлинка изображения будут доступны по адресу `/storage/...`
